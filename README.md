# Job Market Analysis

## Project Objective
This project analyzes job market trends for data science related roles using a dataset containing job titles, salaries, companies, locations, industries, and skill requirements.

## Tasks Performed
- Data extraction from MySQL database
- Data cleaning and preprocessing
- States with most jobs
- Salary analysis by state
- Top industries and companies
- Job title analysis
- Skills required by companies
- Salary vs education analysis
- Insight generation and reporting
- Visualization using Python
- Dataset export for Tableau / Power BI

## Tools Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- MySQL
- GitHub

## Outputs
- Cleaned dataset
- Analysis tables
- Charts
- Insights report
- Power BI / Tableau ready file

## Files Included
- `PRDA_04_Job_Market_Analysis.ipynb`
- `data/processed/job_data_cleaned.csv`
- `outputs/charts/`
- `outputs/tables/`
- `outputs/reports/insights_report.txt`

## Conclusion
The project identifies hiring trends, salary patterns, required skills, and market demand in data science related jobs.
